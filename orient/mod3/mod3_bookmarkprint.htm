Setting Bookmarks for Web Pages

Creating links to Web Pages will allow you to quickly go back to the page at a later time without having to remember or enter the URL.

 

Instructions for Internet Explorer Users

Left click on Favorites. This is located near the top of your Internet Explorer Window.

 

Left click on Add to Favorites.

 

Left click on the OK button that is on the pop-up window that will appear.

 

When you want to visit a Web Page that you have created a bookmark for:

 

Left click on Bookmarks in Firefox or Favorites in Internet Explorer.


A drop down menu will appear. Left click on the link of the web page that you want to visit. When viewing the list of links, you will not see the URL to the web page; you will see a short description of the web page.
