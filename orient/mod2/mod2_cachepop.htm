Set Cache to Always Reload a Page

Set Browser Cache

Firefox 2.0 and higher

  • Select Tools menu > Options... > Advanced > Network tab
  • Set the Cache to 50 MB
  • Click the button
  • Select the Every Time radio button at the bottom
  • Click OK

Internet Explorer 6

  • Select Toolsmenu > Internet Options
  • Select the General tab
  • Select button in the Temporary Internet files section
  • Select the Every Visit to the Page radio button
  • Set the folder size to 50 MB
  • Click OK
  • Still on the General tab, click the button to clear your cache
  • Click OK

Internet Explorer 7

  • Select Toolsmenu > Internet Options
  • Select the General tab
  • Select button in the Browsing history section
  • Select the Every Visit to the Page radio button
  • Set the "Disk space to use" to 50
  • Click OK
  • Still on the General tab, click the button, then click the button to clear your cache
  • When it is finished, click the button
  • Click to close the Internet Options control panel

      2. then on the button.
